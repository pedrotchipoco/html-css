Vamos criar fontes em CSS

font-family

esta fonte serve para dizer qual é o tipo de letra que queremos usar. como exemplo de um font-family temos o ARIAL.É uma fonte bastante usada em em site, posso dizer que não tem nenhum site que não tem esta fonte, pois não vamos usar no nosso site um tipo de letra que não seja coerente com o nosso projeto.

font-size

esta fonte serve para colocar o tamanho do nosso texto, é uma fonte qu é tambem bastante usada nas nossas folhas de estilo pois ela nos ajuda a calibrar o tamanho do nosso texto de forma a deixar o nosso texto mais responsivo para qualquer dispusitivo, desde celulares até a televisores

font-style

serve para dizer se queremos o texto em italico ou se queremos que fica normal

font-weigth

É o peso da fonte, temos como exemplo o lighter que é mais leve e tambem temos o bolder que é mais negrito que o negrito, e outros que não vou citar

text-decoration

serve para sublinhar o nosso site ou para dar enfase. esta fonte
